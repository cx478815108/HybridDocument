# label组件

> label组件是高性能的文本显示组件

使用方法

```
<label style="width: 100%;height: auto;background-color: rgb(20,120,130);font-size: 14px"" @click="buttonClick">{{name}}
</label>
```

## 支持设置的`Native`属性

| 属性 key         | 可选值                                           | 描述                              |
| ---------------- | ------------------------------------------------ | --------------------------------- |
| line-clamp       | 数字类型，单位:无 ,例如：2                       | 文本行数                          |
| color            | 字符串类型，接受rgb,rgba,16进制                  | 文本颜色                          |
| font-size      | 例如：14px                                                       | 字号                                   |
